# [Workstream Name] — Open Questions

Tracker for unresolved decisions. Each question gets a number for easy
reference ("Q3 still blocking us?"). Resolved questions stay in the file —
edit the status, don't delete — so the resolution trail is preserved.

## Format

```
## Q1 — Short question title

**Question:** What needs to be decided?

**Why it matters:** What's blocked or at risk while this is open.

**Status:** [open | resolved | abandoned]

**Resolution:** (filled in when resolved — link to the decisions.md entry)
```
